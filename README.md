An automated security intelligence tool that streamlines threat analysis by orchestrating multiple Large Language Models.

## How it Works

1. **Gemini 3.1 Flash:** Ingests long-form threat reports and extracts key Indicators of Compromise (IOCs).
2. **NVIDIA NIM (Mistral-Large):** Analyzes the extracted intelligence to provide risk scoring and industry impact assessment.
3. **Streamlit:** Provides a clean dashboard for security analysts to triage threats quickly.

## Key Features

* 🔍 Automated IOC extraction from unstructured threat reports
* 📊 AI-generated risk scoring with industry impact assessment
* ⚡ Real-time triage dashboard for security analysts
* 🤖 Multi-LLM orchestration pipeline using Gemini and NVIDIA NIM

## Use Case

Security analysts often need to review long-form threat reports and quickly identify actionable intelligence. This assistant helps reduce manual review time by extracting IOCs, summarizing threat context, assigning a risk score, and presenting the results in an analyst-friendly dashboard.

For example, given a detailed threat report, the tool can identify relevant indicators, assess potential business impact, and provide a triage-ready summary for faster decision-making.
