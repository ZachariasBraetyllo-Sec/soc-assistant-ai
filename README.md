An automated security intelligence tool that streamlines threat analysis by orchestrating multiple Large Language Models.

## How it Works

1. **Gemini 3.1 Flash:** Ingests long-form threat reports and extracts key Indicators of Compromise (IOCs).
2. **NVIDIA NIM (Mistral-Large):** Analyzes the summary to provide high-precision risk scoring and industry impact.
3. **Streamlit:** Provides a clean dashboard for security analysts to triage threats quickly.

🔍 Automated IOC extraction from unstructured threat reports
📊 AI-generated risk scoring with industry impact assessment
⚡ Real-time triage dashboard for security analysts
🤖 Multi-LLM orchestration pipeline (Gemini + NVIDIA NIM)
